# Modul183 — Functional Programming Exercises

This repository contains the exercises and solutions used in the Modul183 (Functional Programming) course. It is intended as a growing collection of hands-on examples, exercises and reference code produced during the class.

Purpose
- Collect and organize exercise files and example implementations in F#.
- Provide runnable sample code for learning and review.
- Evolve gradually as new exercises and solutions are added from lectures and assignments.

Repository structure (high level)
- `Exercise*.fs` — exercise modules and solutions (F# source files).
- `functions/` — shared helper modules used across exercises.
- `Program.fs` — program entry point that can run exercises.
- `exercises.fsproj` — project file to build the solution.

How to use
1. Build and run the project using the .NET SDK (e.g. `dotnet build` and `dotnet run`).
2. Open individual `.fs` files to inspect implementations and examples.

Contributing
- New exercises and solutions from the course will be added over time.
- Prefer small, self-contained modules named `ExerciseX.fs`.
- Keep examples focused and add brief comments where helpful.
